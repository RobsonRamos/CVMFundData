# CVM Data Funds

CVM Data Funds is a project to help you get historical funda data from the CVM (Brazilian SEC entity).


## How to execute the code


```python
   $ docker-compose build
   $ docker-compose up
```

## How to execute the tests 

```python
   $ cd src/
   $ python3 ...

```
